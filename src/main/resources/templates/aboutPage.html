About PC Shop

This is a backend manager of shop Stocks that you can build DIY Desktop PC Tower.
accessory not include (e.g. mouse, keyboard, monitor, etc)

This is a simplified version of PC-shop management system, actual Angular frontend is not included.
The Web-service is coded based on Java Spring Framework with built-in H2 Sql database.
The service is host on AWS Beanstalk with public access.
The project is mainly focused on Backend practice, using minimum front end elements.
